# AI Resume ATS Score Checker

An AI-powered web application that evaluates resumes against job descriptions and provides ATS scores with improvement suggestions.

##  Features
- Upload resume (PDF)
- Extract text using PyPDF2
- Keyword matching with job description
- ATS score calculation
- AI suggestions using Gemini API

##  Tech Stack
- Python
- Flask
- Google Gemini API
- NLP
- HTML/CSS

## Future Improvements
- Better NLP using spaCy

- Login system
